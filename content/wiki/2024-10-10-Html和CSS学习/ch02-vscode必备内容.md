# 2. vscode必备内容

## 2.1 标签分类

标签有双标签和单标签：双标签＜html＞＜/html＞两者是成对存在的，分为开始标签和结束标签；单标签如：＜br /＞。

## 2.2 快速生成HTML骨架

使用 VSCode 时，基本结构不用一个一个输入，可以输入 `!` 然后按 `Tab`，快速自动生成基本结构。

## 2.3 预览插件与实时预览

VSCode 预览页面可右键选择 Open In Default Browser，需要安装 open in browser 插件。

Live Server实时预览插件，在修改标签后看网页界面时，不需要手动刷新就可以实现自动刷新

Live Server必须在vscode打开的文件夹中的文件程序中才能使用

## 2.4 视图缩放

缩放视图 `ctrl+` 可实现代码增大。

`ctrl-` 可实现代码减小。

## 2.5 新建与保存快捷键

VSCode 新建文件快捷键为 `ctrl+N`。

保存快捷键为 `ctrl+S`，将文本保存为 `.html` 文件。

## 2.6 Auto Rename Tag

VSCode 中 Auto Rename Tag 双击前标签，改动之后，后标签都会同步修改。

## 2.7 vscode-icons

vscode-icons 可以实现不同后缀图标显示，便于快速识别文件类型。

## 2.8 Easy LESS

Easy LESS 插件可以编译 less 文件。我们写的 less 不能直接引入到 html 文件中，通过这个插件可以自动生成 css 文件。

## 2.9 会了吧插件

会了吧插件可以展示代码中英语单词的意思。

## 2.10 vscode常用快捷键

①快速复制:shift+alt+下箭头(上箭头）或者ctrl+C然后ctrl+V

②选定多个相同的单词:ctrl+D

③添加多个光标:ctrl+Alt+上箭头(下箭头）

④全局替换某些单词:ctrl+h

⑤快速定位到某一行:ctrl+g

⑥选择某一区块:按住Shift+alt然后拖动鼠标
