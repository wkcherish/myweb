# 1.安装

numpy是科学计算基础库,底层是C语言

三种方式：

## 1.1 通过pip安装

```python
pip install numpy
```

首选方式，自动处理依赖项

在提示系统命令符中使用cmd

## 1.2 通过Anaconda安装

```python
conda install numpy
```

自动解决环境依赖

## 1.3 通过系统包管理器（linux/macos）

对于Debian/Ubuntu

```python
sudo apt-get install python3-numpy
```

对于Fedora:

```python
sudo dnf install python3-numpy
```
